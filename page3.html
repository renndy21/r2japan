PELAJARAN 3

Pada materi ini, terdapat beberapa kanji yang berkaitan dengan alam.
YAMA / やま

goresan : 3
arti             : gunung
kunyomi : yama
onyomi   : san

kotoba :　
山脈 sanmyaku / jarak gunung
富士山 fujisan / gunung fuji
山登り yamanobori / mendaki gunung
山場 yamaba/ klimaks

KAWA / かわ

goresan   : 3
arti              : sungai
kunyomi : kawa
onyomi   : sen

kotoba :
川口 kawaguchi / mulut sungai
小川 ogawa / anak sungai
河川 kasen / sungai

ISHI / いし

goresan  : 5
arti             : batu
kunyomi : ishi
onyomi   : seki,shaku

kotoba :
宝石 houseki / batu mulia
磁石 jishaku / magnet
石油 sekiyu / minyak tanah
化石 kaseki / fosil

TA  / た

goresan  : 5
arti             : ladang
kunyomi : ta
onyomi   : den

kotoba :
水田 suiden / sawah
田園 den'en / area pedesaan
田植え taue / menanam padi
田中 tanaka / nama orang jepang

HANA / はな

goresan : 7
arti             : bunga
kunyomi : hana
onyomi   : ka

kotoba :　
花火 hanabi / kembang api
花粉 kafun / tepung sari
花瓶 kabin / pot bunga
花見 hanami / melihat bunga mekar

AME / あめ

goresan  : 8
arti             : hujan
kunyomi : ame
onyomi   : u

kotoba :
雨雲 amagumo / awan hujan
風雨 fuuu / hujan berangin
雨期 uki / musim hujan
大雨 ooame / hujan lebat

TAKE / たけ

goresan  : 6
arti             : bambu
kunyomi : take
onyomi   : chiku

kotoba :　
竹林 chikurin / hutan bambu
竹刀 shinai / tombak bambu
竹馬 takeuma / tiang
竹垣 takegaki / pagar bambu
