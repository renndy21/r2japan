PELAJARAN 6

Pada materi ini, terdapat beberapa kanji yang berkaitan dengan sekolah.
GAKU / がく

goresan  : 8
arti             : belajar
kunyomi : mana-bu
onyomi   : gaku

kotoba :　
大学 daigaku / universitas
学校 gakkou / sekolah
文学 bungaku / sastra
入学 nyuugaku / masuk sekolah
学ぶ manabu / belajar

KOU / こう

goresan : 10
arti             : sekolah
kunyomi : -
onyomi   : kou

kotoba :
学校 gakkou / sekolah
校舎 kousha / bangunan sekolah
校則 kousoku / peraturan sekolah
校正 kousei / koreksi

SAKI / さき

 goresan : 6
arti             : ujung
kunyomi : saki
onyomi    : sen

kotoba :
先月 sengetsu / bulan lalu
指先 yubisaki / ujung jari
先輩 senpai / senior
先週 senshuu / minggu lalu

UMU / うむ

goresan   : 5
arti            : kehidupan
kunyomi : u-mu,i-kiru
onyomi   : sei,shou

kotoba :　
生意気 namaiki / lancang
学生 gakusei / pelajar
生年月日 seinengappi / tanggal lahir
生まれる umareru /  dilahirkan
長生き nagaiki / 
panjang umur

NA / な

goresan  : 6
arti             : nama
kunyomi : na
onyomi   : mei, myou

kotoba :　
名作 meisaku / masterpiece
名詞 meishi / kata benda
名前 namae / nama
名字 myouji / nama keluarga

JI / じ

goresan  : 6
arti             : huruf
kunyomi : aza
onyomi   : ji

kotoba :
漢字 kanji / huruf han
文字 moji / 
huruf(karakter)
数字 suuji / bilangan
十字 juuji / tanda silang (tambah)

HON / ほん

goresan  : 5
arti            : buku
kunyomi : moto
onyomi   : hon

kotoba :　
本物 honmono / asli
本屋 honya / toko buku
本棚 hondana / rak buku
基本 kihon / dasar

KARADA / からだ

goresan : 7
arti             : tubuh
kunyomi : karada
onyomi   : tai

kotoba :　
体 karada / tubuh
体育 taiiku / 
pendidikan
体温 taion / suhu tubuh
体重 taijuu / berat badan
