PELAJARAN 15

Pada materi dibawah ini, merupakan materi kanji yang berkaitan dengan kata sifat.
ATARA-SHII / あたらーしい

goresan  : 13
arti             : baru
kunyomi : atara-shii, ara-ta
onyomi  : shin

kotoba :　
新聞 shinbun / koran
新しい atarashii / baru
新た arata / segar
新年 shinnen /  tahun baru

FURU-I / ふるーい

goresan : 5
arti             : lama/tua
kunyomi : furu-i
onyomi   : ko

kotoba :
古典 koten / klasik
古代 kodai / 
zaman dulu/purba
古里 furusato / 
kota kelahiran
古本 furuhon / buku bekas

NAGA-I / ながーい

goresan : 8
arti             : panjang, kepala/ketua
kunyomi : naga-i
onyomi   : chou

kotoba :　
校長 kouchou / 
kepala sekolah
社長 shachou / 
direktur perusahaan
長生き nagaiki / panjang umur
市長 shichou/ walikota

YASU-I / やすーい

goresan : 6
arti             : murah
kunyomi : yasu-i
onyomi   : an

kotoba :　
安心 anshin / lega
不安 fuan / resah
安全 anzen / keselamatan
安物 yasumono / 
barang murah

MOTO / もと

goresan  : 4
arti             : asal-usul
kunyomi : moto
onyomi   : gen, gan

kotoba :　
元気 genki / sehat
火の元 hinomoto / 
asal api
三次元 sanjigen / 
tiga dimensi
元日 ganjitsu / 
hari di tahun baru

KI / き

goresan  : 6
arti             : mood
kunyomi : -
onyomi   : ki

kotoba :
空気 kuuki / udara
天気 tenki / cuaca
電気 denki / listrik
気分 kibun / mood

OO-I / おおーい

goresan : 6
arti             : banyak
kunyomi : oo-i
onyomi   : ta

kotoba :　
多い ooi / banyak
多分 tabun / mungkin
多数 tasuu / jumlah banyak
多量 taryou / jumlah besar

SUKO-SHI / すこーし

goresan : 4
arti            : sedikit
kunyomi : suko-shi, suku-nai
onyomi   : shou

kotoba :　
少女 shoujo / gadis
少年 shounen / remaja
少し sukoshi / dikit (agak)
少ない sukunai /sedikit
