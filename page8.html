PELAJARAN 7

Pada materi ini, terdapat beberapa kanji yang berkaitan dengan sekolah.
OOKII / おおきい

goresan  : 3
arti             : besar
kunyomi : oo-kii
onyomi   : dai,tai

kotoba :
大空 oozora / 
langit luat
大統領 daitouryou / 
presiden
大きい ookii / besar
大切 taisetsu / penting

CHIISAI / ちさい

goresan : 3
arti             : kecil
kunyomi : chii-sai,ko,o
onyomi   : shou

kotoba :
小犬 koinu / 
anak anjing
小さい chiisai / kecil
小川 ogawa / 
anak sungai
大小 daishou / 
besar dan kecil 

TAKA-I / たかーい

goresan  : 10
arti         : tinggi, mahal
kunyomi : taka-i
onyomi   : kou

kotoba :　
高空 takazora / 
ketinggian langit
高級 koukyuu / 
level tinggi
高校 koukou / SMA
円高 endaka / 
kenaikan yen

TOMO / とも

goresan : 4
arti             : teman
kunyomi : tomo
onyomi   : yuu

kotoba :　
友情 yuujou / 
pertemanan
友達 tomodachi / 
teman
親友 shinyuu / teman dekat
盟友 meiyuu / teman sejati

I-RERU / いーれる

goresan   : 2
arti             : masuk
kunyomi : hai-ru, i-reru
onyomi   : nyuu

kotoba :
入力 nyuuryoku / input
入学 nyuugaku / masuk sekolah
入ります hairimasu / masuk
入れます iremasu / memasukkan

DE-RU / でーる

goresan  : 5
arti             : keluar
kunyomi : de-ru,da-su
onyomi   : shutsu

kotoba :
出口 deguchi / pintu keluar
出発 shuppatsu / keberangkatan
出来る dekiru / bisa
思い出す omoidasu / mengingat

MON / もん

goresan : 9
arti             : gerbang
kunyomi : kado
onyomi   : mon

kotoba :
専門 senmon / spesialisasi
門出 kadode / berangkat
門番 monban / penjaga gerbang
正門 seimon / gerbang utama
