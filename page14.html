PELAJARAN 13

Pada materi dibawah ini, merupakan materi kanji yang berkaitan dengan arah mata angin.
MACHI / まち

goresan : 7
arti             : kota
kunyomi : machi
onyomi   : chou

kotoba :
町長 chouchou / 
walikota
町名 choumei / nama kota
町役場 machiyakuba / 
balai kota
港町 minatomachi / 
kota pelabuhan

TERA / てら

goresan : 6
arti             : kuil
kunyomi : tera
onyomi   : ji

kotoba :　
東大寺 toudaiji / kuil toudai
山寺 yamadera / kuil gunung

KURUMA / くるま

goresan : 7
arti             : mobil
kunyomi : kuruma
onyomi    : sha

kotoba :
車輪 sharin / roda
自転車 jitensha / 
sepeda
電車 densha / 
kereta
車椅子 kurumaisu / 
kursi roda

DEN / でん

goresan : 13
arti             : listrik
kunyomi : -
onyomi   : den

kotoba :
電池 denchi / baterai
電車 densha / kereta 
電話 denwa / telepon
電気  denki / listrik

NISHI / にし

goresan  : 6
arti             : barat
kunyomi : nishi
onyomi   : sei,sai

kotoba :　
関西 kansai / 
jepang bagian barat
西瓜 suika/semangka
西洋 seiyou / samudra
大西洋 daiseiyou / 
samudra atlantis

KITA / きた

goresan : 5
arti             : utara
kunyomi : kita
onyomi   : hoku

kotoba :　
東北 touhoku / timur laut
北極 hokkyoku / kutub utara
北欧 hokuou / eropa utara
北半球 kitahankyuu / dunia belahan

HIGASHI / ひがし

goresan : 8
arti             : timur
kunyomi : higashi
onyomi   : tou

kotoba :
東京 toukyou / tokyo
東北 touhoku / timur laut
中東 chuutou / timur tengah
東半球higashihankyuu/
belahan bumi bagian timur

MINAMI / みなみ

goresan : 9
arti             : selatan
kunyomi : minami
onyomi   : nan

kotoba :
東南 tounan/ tenggara
南風 minamikaze / angin selatan
南半球 minamihankyuu / 
bumi belahan selatan
南極 nankyoku / 
kutub selatan
