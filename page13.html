PELAJARAN 12

Pada materi dibawah ini, merupakan materi kanji yang berkaitan dengan melakukan aktivitas.
I-U / いーう

goresan : 7
arti             : berkata
kunyomi : koto, i-u
onyomi   : gen, gon

kotoba :　
言葉 kotoba / berkata
言語 gen'go / bahasa
方言 hougen / bahasa daerah
言う iu / berkata

KA-KU / かーく

goresan  : 10
arti             : menulis
kunyomi : ka-ku
onyomi   : sho

kotoba :　
絵葉書 ehagaki / 
kartu post bergambar
書く kaku / menulis
著書 chosho / buku sastra
辞書 jisho / kamus

HANA-SU / はなーす

goresan  : 13
arti           : berbicara
kunyomi : hanashi, hana-su
onyomi   : wa

kotoba :
会話 kaiwa / percakapan
電話 denwa / telepon
昔話mukashibanashi / dongeng
実話 jitsuwa / kisah nyata

TABE-RU / たべーる

goresan  : 9
arti             : makan
kunyomi : tabe-ru,
ku-u
onyomi   : shoku, jiki

kotoba :
食べる taberu / 
makan
食事 shokuji / 
makanan
断食 danjiki / puasa
食欲 shokuyoku / 
nafsu makan

NO-MU / のーむ

goresan : 12
arti             : minum
kunyomi : no-mu
onyomi   : in

kotoba :　
飲食 inshoku / makan minum
飲む nomu / minum
飲酒 inshu / minum arak
飲み物 nomimono / minuman

GO / ご

goresan : 14
arti             : bahasa
kunyomi : kata-ru
onyomi   : go

kotoba :　
日本語 nihongo / bahasa jepang
物語 monogatari / kisah
語る kataru / berucap
外国語 gaikokugo / bahasa asing

KA-U / かーう

goresan  : 12
arti             : membeli
kunyomi : ka-u
onyomi   : bai

kotoba :　
買い物 kaimono / belanja
買う kau / membeli
買収 baishu / suap
売買 baibai / jual beli
