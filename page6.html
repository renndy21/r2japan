PELAJARAN 5

Pada materi ini, terdapat beberapa kanji yang berkaitan dengan arah dan letak.
UE / うえ

goresan  : 3
arti             : atas
kunyomi : ue,a-garu
onyomi   : jou

kotoba :　
召し上がる meshiagaru / makan
屋上 okujou / atap
上達 joutatsu / kemajuan
上手 jouzu / pandai

SHITA / した

goresan  : 3
arti             : bawah
kunyomi : shita, kuda-saru, sa-garu
onyomi   : ka, ge

kotoba :
下落 geraku / merosot
下手 heta / payah
下着 shitagi / pakaian dalam
上下 jouge / atas bawah

HIDARI / ひだり

goresan  : 5
arti             : kiri
kunyomi : hidari
onyomi    : sa

kotoba :
左折 sasetsu / belok kiri
左翼 sayoku / sayap kiri
左側 hidarigawa / sebelah kiri

MIGI / みぎ

goresan : 5
arti             : kanan
kunyomi : migi
onyomi   : u, yuu

kotoba :
右手 migite / tangan kanan
右翼 uyoku / sayap kanan
左右 sayuu / kiri kanan (kontrol)

SOTO / そと

goresan : 5
arti             : di luar
kunyomi : soto, hoka
onyomi   : gai, ge

kotoba :　
外人 gaijin / orang asing
外国 gaikoku / negara luar
外出 gaishutsu / pergi keluar
外科医 gekai / dokter bedah

UCHI / うち

goresan  : 4
arti             : bagian dalam
kunyomi : uchi
onyomi   : nai

kotoba :
内容 naiyou / konten/isi
内戦 naisen / perang sipil
内部 naibu / interior
家内 kanai / istri

NAKA / なか

goresan : 4
arti             : tengah
kunyomi : naka
onyomi   : chuu

kotoba :
中心 chuushin / pusat (center)
中年 chuunen / pertengahan tahun
中学校 chuugakkou / SMP
中指 nakayubi / jari tengah
